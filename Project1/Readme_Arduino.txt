Editor Used : Arduino
Board Used: Arduino Genuino/Uno

Selecting the board and port:
1. Connect the arduino to laptop using serial communication port.
2. Select arduino uno from Tools>Board>Arduino UGenuino/Uno.
3. Select the valid COM port from Tools>Port.

Compiling code:
1. Compile the code by clicking on the Verify button.

Uploading code:

4. Upload the code to the atmega328 on the arduino board by clicking on the upload button.

This code, samples the analog values from the LDR and Potentiometer and writes on the serial communication port.
